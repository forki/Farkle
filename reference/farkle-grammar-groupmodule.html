Functions to work with Groups.

Functions and values

Function or valueDescription
getSymbolGroup groups x
Signature: groups:RandomAccessList<Group> -> x:Symbol -> Group option

Like getSymbolGroupIndexed, but returns a Group, instead of an index. Such group might not exist; in this case, None is returned.
getSymbolGroupIndexed groups x
Signature: groups:seq<Group> -> x:Symbol -> Indexed<Group> option

Gets the index of the group in a list tha has the specified symbol either its start, or end, or container symbol. Such index might not exist; in this case, None is returned.
