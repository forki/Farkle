Farkle

Parser

Namespace: Farkle.Parser

A type signifying the state of a parser. The parsing process can be continued by evaluating the lazy Parser values. This type is the lowest-level public API. It is modeled after the Designing with Capabilities presentation.

Union Cases

Union CaseDescription
Continuing(...)
Signature: Position * ParseMessage * Lazy<Parser>

The parser has completed one step of the parsing process. The log message of it is returned as well as a thunk of the next parser.
Failed(Position,ParseMessage)
Signature: Position * ParseMessage

The parser has failed. No lazy parser is returned, so the parsing process cannot continue.
Finished(Position,Reduction)
Signature: Position * Reduction

The parser has finished parsing. No lazy parser is returned, as the parsing process is complete.
Started(Lazy<Parser>)
Signature: Lazy<Parser>

The parser has just started.
